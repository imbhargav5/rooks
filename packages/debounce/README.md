### Installation

```
npm install --save @rooks/use-debounce
```

### Importing the hook

```javascript
import useDebounce from "@rooks/use-debounce"
```

### Usage

```jsx
function Demo() {
  useDebounce();
  return null
}

render(<Demo/>)
```
