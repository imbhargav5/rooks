## About
Drop in replacement for javascript console but it is reactive.


[//]: # (Main)

## Installation

```
npm install rooks
```

## Importing the hook

```javascript
import {useConsole} from "rooks"
```

## Usage

```jsx
function Demo() {
  useConsole();
  return null
}

render(<Demo/>)
```
