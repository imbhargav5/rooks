## About

useState but syncs with sessionstorage

[//]: # "Main"

## Installation

    npm install --save rooks

## Importing the hook

```javascript
import {useSessionstorageState} from "rooks"
```

## Usage

```jsx
function Demo() {
  useSessionstorageState();
  return null
}

render(<Demo/>)
```
