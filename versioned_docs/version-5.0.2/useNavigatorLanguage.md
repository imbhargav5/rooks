## About

Navigator Language hook for React.

## Installation

    npm install --save rooks

## Importing the hook

```javascript
import {useNavigatorLanguage} from "rooks";
```

## Usage

```jsx
function Demo() {
  const language = useNavigatorLanguage();
  return <p>Language is {language}</p>;
}

render(<Demo />);
```

### Return value

A language (String) is returned.
