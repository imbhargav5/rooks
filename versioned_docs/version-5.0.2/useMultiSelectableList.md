## About

A custom hook to easily select multiple values from a list

[//]: # "Main"

## Installation

    npm install --save rooks

## Importing the hook

```javascript
import {useMultiSelectableList} from "rooks"
```

## Usage

```jsx
function Demo() {
  useMultiSelectableList();
  return null
}

render(<Demo/>)
```
